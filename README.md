# tester-skills

一个测试工程师技能共享社区项目。收集、整理、共享 AI IDE 测试相关技能，支持团队协作和开源共建。

## 支持的平台

CodeBuddy、Cursor、Claude Code、Trae 等主流 AI IDE

## 项目结构

```
tester-skills/
├── skills/              # 技能目录
│   ├── testcase-generator/
│   └── testcase-review/
├── README.md
├── .gitignore
└── LICENSE
```

## 已有技能

| 技能名称 | 说明 |
|----------|------|
| testcase-generator | 根据 PRD 文档自动生成功能测试用例 |
| testcase-review | 对比测试用例与需求文档，找出缺失 |
