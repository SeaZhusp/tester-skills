## 二、高流量选题方向

### 1. 薪资&职业发展类（流量王者）

**选题模式**：
- AI测试工程师薪资大揭秘
- 面试真题+答案解析
- 简历优化指南
- 跳槽避坑指南

**标题模板**：
- "AI测试工程师薪资调研报告（2024最新版）"
- "面试了50个测试工程师，我发现薪资高的人都有这个特点"

### 2. 工具对比评测类（实用性强）

**选题模式**：
- XX工具深度评测
- XX vs XX全面对比
- XX工具红黑榜

**标题模板**：
- "ChatGPT vs Copilot vs Claude：AI测试哪家强？"
- "我用了一年的AI测试工具，给你一个真实的红黑榜"

### 3. 实战案例类（转发率高）

**选题模式**：
- 真实项目案例
- 问题解决全过程
- 踩坑+爬坑记录

**标题模板**：
- "甲方要求3天完成全链路测试，我用AI工具做到了"
- "线上事故后，我用AI测试重构了整个流程"

### 4. 入门科普类（受众最广）

**选题模式**：
- 小白入门指南
- 概念扫盲
- 学习路径规划

**标题模板**：
- "测试工程师学AI，看这一篇就够了"
- "AI测试是什么？一文解释清楚"

### 5. 趋势洞察类（行业影响力）

**选题模式**：
- 行业报告解读
- 大厂技术揭秘
- 未来趋势预测

**标题模板**：
- "Google/微软/苹果的测试团队都在用什么AI工具"
- "AI测试5年后会取代手工测试吗？"

## 三、选题评估模型

### AIPLUS模型

| 维度 | 评估问题 | 最低标准 |
|------|----------|----------|
| A-Awareness | 话题有热度吗？ | 微信指数>5000 |
| I-Interest | 读者会感兴趣吗？ | 目标人群>10万 |
| P-Profit | 对我有利吗？ | 能涨粉或变现 |
| L-Learnable | 读者能学到东西吗？ | 有具体知识点 |
| U-Usable | 有实操价值吗？ | 能落地执行 |
| S-Shareable | 读者会转发吗？ | 有社交货币 |

### 选题否决线

以下情况建议放弃：
- 话题过于小众（受众<1万）
- 已经烂大街（竞品已发10篇+）
- 时效性已过（热点降温>2周）
- 风险不可控（涉及敏感话题）
- 难以写出新意（没有独特角度）

## 四、选题执行清单

```
□ 1. 确定选题大类（薪资/工具/案例/入门/趋势）
□ 2. 搜索微信指数验证热度
□ 3. 搜索竞品文章看是否有新角度
□ 4. 确定独特切入角度
□ 5. 构思3种不同标题方案
□ 6. 评估内容能否支撑标题承诺
□ 7. 确认发布时间（热点余温期）
□ 8. 规划系列文章（如有）
```

## 六、选题黑名单

以下话题不建议做：
- 涉及具体公司负面（法律风险）
- 政治敏感话题
- 竞品恶意对比
- 过时热点（强行蹭）
- 纯技术无干货（水文）
